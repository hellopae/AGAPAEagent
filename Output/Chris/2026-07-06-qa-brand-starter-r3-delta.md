ถ้อยคำใหม่ "ธรรมเนียมเดิม ม่วง — ไม่ต้องคู่กับสีใด — เป็นสีไว้ทุกข์เฉพาะของแม่ม่าย (ผู้ไว้ทุกข์คนอื่นใส่ดำแยกกัน)" ตรงกับผล fact-check C4 ของ Reese แล้ว — การกลับความหมายใน r2 ("เฉพาะเมื่อคู่กับดำ") ถูกลบออกหมด และประโยคปิดท้าย ("ม่วงเดี่ยว ๆ ยังมีน้ำหนักไว้ทุกข์อยู่บ้าง ... ใช้เฉพาะบริบทวันเสาร์/งานเฉลิมพระเกียรติที่ระบุชัดเท่านั้น ไม่ใช้แบบลอย ๆ") ให้แนวปฏิบัติที่ตีความทางเดียว — ปลอดภัยต่อการใช้เป็น baseline

**2. W1 — ขอบเขตกติกาสี 3.3(3) (บรรทัด 102): ✅ หายคลุมเครือ**
แยก "สัญลักษณ์ศักดิ์สิทธิ์หลัก" (ธรรมจักร, เจดีย์ ฯลฯ → ทองบุญ/น้ำตาลไม้สักเท่านั้น) ออกจาก "ดอกบัวประกอบตกแต่ง" (ใช้ชมพูบัวได้ตามข้อ 1) ชัดเจน — ความขัดแย้งระหว่างข้อ 1 กับ 3.3(3) หายไป กรณีก้ำกึ่งที่เหลือ (เช่น ใบโพธิ์นับเป็น "หลัก" ไหม) มีทางออกอยู่แล้วในข้อ 3.3(4) escalate ผ่าน Claudy → Reese จึงไม่ค้างเป็นช่องตีความ

**3. W2 — หมายเหตุ PNG/CMYK (บรรทัด 76): ✅ เพียงพอ**
ระบุครบทั้งเหตุ (PNG alpha = RGB ในทางปฏิบัติ — Reese VERIFIED แล้วว่า PNG spec ไม่มีโหมด CMYK+alpha), ขอบเขตใช้งาน (design/mockup/เว็บ), และข้อบังคับ (ไฟล์เข้าสายพิมพ์ flatten เป็น CMYK เสมอ) — สอดคล้อง SOP-08 ข้อ D และปิดช่องเข้าใจผิดว่า PNG คือไฟล์พร้อมพิมพ์

**4. W3 — วงเล็บนิยามใน prompt block (บรรทัด 71): ✅ ใช้ได้**

**5. W4 — ระบบปี (ทั้งฉบับ): ✅ สอดคล้องแล้ว**
