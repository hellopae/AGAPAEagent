5 ความผิดพลาดที่ทำให้งานพิมพ์ออกมาไม่ตรงปก (และวิธีป้องกัน)


ในช่วง 40 กว่าปีที่ TANAPAT Printing รับงานพิมพ์มา ความผิดพลาดส่วนใหญ่ไม่ได้เกิดจากโรงพิมพ์ มันเกิดก่อนที่ไฟล์จะถึงมือเราด้วยซ้ำ บทความนี้เขียนเพื่อให้รู้ว่าควรเช็กอะไรบ้าง ก่อนที่งบประมาณจะหายไปกับงานพิมพ์ที่ใช้ไม่ได้


1. ส่งไฟล์ RGB แทน CMYK — สีเพี้ยนทันที

จอคอมแสดงสีในระบบ RGB แต่เครื่องพิมพ์ทำงานด้วยหมึก CMYK ทั้งสองระบบมี gamut สีที่ต่างกัน สีสดบนจออย่างฟ้าจัด เขียวนีออน หรือม่วง พอพิมพ์ออกมามักหม่นลงหรือเปลี่ยนโทนไปเลย

วิธีป้องกัน ตั้งค่า Color Mode เป็น CMYK ตั้งแต่เปิดไฟล์ใหม่ ไม่ใช่แปลงทีหลัง ใน Illustrator เช็กหรือเปลี่ยนได้ที่ File > Document Color Mode ส่วน InDesign ไม่มี Color Mode ระดับเอกสาร ให้กำหนดสีใน Swatches เป็น CMYK และเลือกแปลงสีเป็น CMYK ตอน Export PDF ได้อีกชั้น ใน Photoshop เช็กที่ Image > Mode


2. ไม่ทำ Bleed — ขอบขาวโผล่หลังตัด

เครื่องตัดกระดาษไม่ได้ตัดตรงร้อยเปอร์เซ็นต์ มีการเหลื่อมเล็กน้อยเสมอ ถ้าสีหรือภาพในไฟล์ชิดขอบพอดี พอตัดออกมาจะเห็นขอบกระดาษขาวทันที Bleed คือพื้นที่กันชนที่ต้องยืดสีหรือพื้นหลังออกจากขอบจริงอีก 3 มิลลิเมตร

วิธีป้องกัน ทุกงานที่มีสีหรือภาพพาดถึงขอบ ให้ตั้ง Bleed และยืดพื้นหลังให้เต็ม Bleed Area เสมอ


3. ความละเอียดต่ำเกินไป (72 dpi แทน 300 dpi) — ภาพเบลอ

ภาพที่คมชัดบนจอไม่ได้แปลว่าจะคมชัดเมื่อพิมพ์ ไฟล์ภาพสำหรับเว็บหรือโซเชียลมักถูกบันทึกที่ความละเอียดต่ำเพื่อให้โหลดเร็ว แต่งานพิมพ์ต้องการอย่างน้อย 300 dpi ที่ขนาดจริงของงาน ภาพที่ดึงจากเว็บหรือ Facebook จึงมักละเอียดไม่พอ พิมพ์ออกมาจะเบลอและเห็นพิกเซล

วิธีป้องกัน เช็ก Resolution ก่อนวางภาพในไฟล์งานทุกครั้ง ใน Photoshop ดูได้ที่ Image > Image Size ถ้าต่ำกว่า 300 dpi ให้หาไฟล์ต้นฉบับที่ใหญ่กว่า หรือใช้ภาพ Vector แทน


4. Font เปลี่ยนเองเมื่อเปิดบนเครื่องอื่น — Layout พังทั้งหน้า

ไฟล์ที่ส่งไปไม่ได้พาฟอนต์ไปด้วยโดยอัตโนมัติ เมื่อโรงพิมพ์เปิดบนเครื่องที่ไม่มีฟอนต์ตัวเดียวกัน ระบบจะเปลี่ยนฟอนต์เป็นตัวอื่นและ Layout พังทันที

วิธีป้องกัน มีสองทาง หนึ่ง Outline Font แปลงตัวอักษรให้กลายเป็น Vector Path ใน Illustrator ใช้ Type > Create Outlines ปลอดภัยที่สุดแต่แก้ข้อความไม่ได้อีก ควรเก็บไฟล์ต้นฉบับไว้ก่อน สอง Export เป็น PDF มาตรฐาน PDF/X-1a หรือ PDF/X-4 ซึ่งจะฝังฟอนต์ทั้งหมดลงในไฟล์ให้อัตโนมัติ ไม่ต้องตั้งค่าอะไรเพิ่ม สำหรับฟอนต์ภาษาไทยเรื่องนี้สำคัญมากเป็นพิเศษ


5. ไม่ตรวจ Proof ก่อนพิมพ์จริง — พลาดแล้วพลาดเลย

งานพิมพ์ไม่ใช่ไฟล์ดิจิทัลที่กด Undo ได้ พิมพ์ออกมาแล้วจบ เราเจอกรณีเบอร์โทรผิด โลโก้ขาดหาย ที่อยู่ไม่ถูก หรือสีโลโก้เพี้ยน ทั้งหมดรู้เมื่อรับงานไปแล้วและต้องพิมพ์ใหม่ทั้งหมด

วิธีป้องกัน ก่อนอนุมัติพิมพ์ทุกครั้ง เช็กว่าข้อความถูกต้องครบถ้วนไหม เบอร์โทร อีเมล เว็บไซต์ถูกต้องไหม สีโลโก้ตรงกับ Brand Guide ไหม ขนาดและทิศทางของงานถูกต้องไหม และให้คนที่ไม่ได้ทำงานนี้ช่วยอ่านทวนด้วย เพราะตาเราเองมักมองข้ามสิ่งที่คุ้นเคย


Checklist ก่อนส่งไฟล์โรงพิมพ์

- Color Mode เป็น CMYK
- ตั้งค่า Bleed 3 มม. ขึ้นไป
- ความละเอียดภาพ 300 dpi ที่ขนาดจริง
- Outline Font หรือ Embed ใน PDF/X
- ตรวจ Proof และให้ผู้อื่นช่วยอ่านทวน


ความผิดพลาดเหล่านี้ป้องกันได้ทั้งหมดก่อนส่งไฟล์ ใช้เวลาไล่เช็กตาม Checklist ข้างบนแค่ไม่กี่นาที ดีกว่าปล่อยให้งบและเวลาหายไปกับงานพิมพ์ที่ใช้ไม่ได้
