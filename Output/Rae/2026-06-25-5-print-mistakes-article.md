# 5 ความผิดพลาดที่ทำให้งานพิมพ์ออกมาไม่ตรงปก (และวิธีป้องกัน)

> **ออกแบบสวยมา 2 สัปดาห์ ส่งโรงพิมพ์แล้วสีเพี้ยน — นี่คือบทเรียนที่แพงที่สุดในวงการดีไซน์ไทย**

---

ในช่วง 40 กว่าปีที่ TANAPAT Printing รับงานพิมพ์มา เราได้เห็นทุกอย่าง ทั้งงานที่ออกมาสวยงามตรงใจลูกค้า และงานที่ต้องพิมพ์ซ้ำเพราะพลาดตั้งแต่ไฟล์

ความจริงที่เราอยากบอกตรงๆ คือ ความผิดพลาดส่วนใหญ่ไม่ได้เกิดจากโรงพิมพ์ — มันเกิดก่อนที่ไฟล์จะถึงมือเราด้วยซ้ำ

บทความนี้ไม่ได้เขียนขึ้นเพื่อโชว์ว่าเราเก่ง แต่เขียนเพื่อให้ทุกคนที่ทำงานออกแบบ ทำสื่อ หรือสั่งพิมพ์งาน รู้ว่าควรเช็กอะไรบ้าง ก่อนที่งบประมาณจะหายไปกับงานพิมพ์ที่ใช้ไม่ได้

---

## 1. ส่งไฟล์ RGB แทน CMYK — สีเพี้ยนทันที

นี่คือความผิดพลาดอันดับหนึ่งที่เราเจอบ่อยที่สุด

จอคอมพิวเตอร์แสดงสีในระบบ RGB (Red, Green, Blue) ซึ่งเป็นระบบสีของแสง แต่เครื่องพิมพ์ทำงานด้วยหมึก CMYK (Cyan, Magenta, Yellow, Black) ทั้งสองระบบมี **Gamut สีที่ต่างกัน** — สีบางเฉดบน RGB เช่น ฟ้าสด, เขียวนีออน, ม่วงจัด ไม่สามารถจำลองได้ด้วยหมึก CMYK ทำให้แปลงแล้วสีเพี้ยน

สีที่สดบนหน้าจอ เช่น น้ำเงินสด ส้มสว่าง หรือเขียวนีออน — พอพิมพ์ออกมามักจะหม่นลง เข้มขึ้น หรือเปลี่ยนโทนไปเลย บางเคสที่เราเจอ สีม่วงบนจอกลายเป็นสีน้ำเงินในงานพิมพ์จนลูกค้าทักทันทีที่เปิดกล่อง

**วิธีป้องกัน:** ตั้งค่า Color Mode เป็น CMYK ตั้งแต่เริ่มเปิดไฟล์ใหม่ ไม่ใช่แปลงทีหลัง หากเป็น Illustrator หรือ InDesign ให้เช็กที่ Document Setup ถ้าใช้ Photoshop ให้ดูที่ Image > Mode ก่อนออกแบบทุกครั้ง

---

## 2. ไม่ทำ Bleed — ขอบขาวโผล่หลังตัด

ลองนึกภาพนี้: พิมพ์บัตรนามบัตรออกมาสวยมาก แต่พอตัดแล้วมีขอบขาวบางๆ โผล่รอบด้าน ทั้งที่ในไฟล์ไม่มีเลย

สาเหตุคือ เครื่องตัดกระดาษไม่ได้ตัดตรงร้อยเปอร์เซ็นต์ มีการเหลื่อมเล็กน้อยเสมอ ซึ่งเป็นเรื่องปกติของกระบวนการผลิต ถ้าสีหรือภาพในไฟล์ของคุณไปชิดขอบพอดี พอตัดออกมาก็จะเห็นขอบกระดาษขาวๆ ทันที

ขอบขาวเส้นเล็กๆ ที่ไม่มีใครอยากเห็นบนนามบัตรวันสำคัญ — นั่นคือผลของการไม่เผื่อ Bleed

**Bleed** คือพื้นที่กันชนที่ต้องทำให้สีหรือพื้นหลังเลยออกมาจากขอบจริงอีก 3 มิลลิเมตร (บางงานใช้ 5 มม.)

**วิธีป้องกัน:** ทุกงานที่มีสีหรือภาพพาดถึงขอบ ต้องตั้ง Bleed ในโปรแกรมออกแบบ และยืดพื้นหลังออกไปให้เต็ม Bleed Area เสมอ โรงพิมพ์ที่ดีจะระบุค่า Bleed ที่ต้องการในใบสั่งงาน อย่าลืมอ่านให้ครบ

---

## 3. ความละเอียดต่ำเกินไป (72 dpi แทน 300 dpi) — ภาพเบลอ

ภาพที่คมชัดบนจอมือถือหรือหน้าจอ Full HD ไม่ได้แปลว่าจะคมชัดเมื่อพิมพ์ลงกระดาษ

จอแสดงผลใช้ความละเอียดที่ 72–96 dpi (dots per inch) แต่งานพิมพ์คุณภาพดีต้องการอย่างน้อย **300 dpi** ที่ขนาดจริงของงาน ถ้านำภาพจากเว็บ จาก Facebook หรือจากไฟล์ที่บีบอัดมาใช้ในงานพิมพ์ ผลลัพธ์ที่ได้คือภาพเบลอ ฝุ่นๆ มองเห็นพิกเซล — โดยเฉพาะงานขนาดใหญ่อย่างโปสเตอร์หรือ Banner

**วิธีป้องกัน:** ก่อนวางภาพในไฟล์งาน ให้เช็ก Resolution ของไฟล์ภาพนั้นก่อนเสมอ ถ้าเป็น Photoshop ดูได้ที่ Image > Image Size ถ้าความละเอียดต่ำกว่า 300 dpi ให้หาไฟล์ต้นฉบับที่ใหญ่กว่า หรือใช้ภาพเวกเตอร์ (Vector) แทน เพราะเวกเตอร์ไม่มีปัญหาเรื่อง dpi

## 4. Font เปลี่ยนเองเมื่อเปิดบนเครื่องอื่น — Layout พังทั้งหน้า

เคยส่งไฟล์ไปแล้วโรงพิมพ์โทรมาบอกว่า "ฟอนต์หาย" ไหม?

เวลาออกแบบงาน ฟอนต์ที่คุณใช้อยู่ในเครื่องคอมพิวเตอร์ของคุณ แต่ไฟล์ที่ส่งไปไม่ได้พาฟอนต์นั้นไปด้วยโดยอัตโนมัติ เมื่อโรงพิมพ์เปิดไฟล์บนเครื่องที่ไม่มีฟอนต์ตัวเดียวกัน ระบบจะเปลี่ยนฟอนต์เป็นตัวอื่นโดยอัตโนมัติ บางทีเปลี่ยนไปเป็น Arial หรือ Times New Roman ซึ่ง Layout ทั้งหน้าพังทันที

**วิธีป้องกัน:** มี 2 ทางเลือก เลือกทางใดทางหนึ่งตามสถานการณ์

- **Outline Font (Create Outlines)** — แปลงตัวอักษรทุกตัวให้กลายเป็น Vector Path ใน Illustrator ใช้ Type > Create Outlines ปลอดภัยที่สุด โรงพิมพ์เปิดเครื่องไหนก็ได้ผลเหมือนกัน ข้อเสียคือแก้ไขข้อความไม่ได้อีก ควรเก็บไฟล์ต้นฉบับก่อน Outline ไว้เสมอ

- **Embed Font** — ฝังข้อมูลฟอนต์ไว้ในไฟล์ PDF โดยตรง ทำได้ตอน Export เป็น PDF เลือก Standard เป็น PDF/X-1a หรือ PDF/X-4 และเปิดตัวเลือก "Embed All Fonts" เหมาะกับงานที่ยังต้องการแก้ไขข้อความได้ในภายหลัง

สำหรับงานที่ใช้ฟอนต์ภาษาไทย เรื่องนี้สำคัญมาก เพราะฟอนต์ไทยมีหลายรุ่น ต่างกันแม้จะชื่อเดียวกัน

## 5. ไม่ตรวจ Proof ก่อนพิมพ์จริง — พลาดแล้วพลาดเลย

งานพิมพ์ไม่ใช่ไฟล์ดิจิทัลที่แก้แล้ว Undo ได้ เมื่อพิมพ์ออกมาแล้ว — จบ

เราเจอกรณีที่น่าเสียใจหลายครั้ง เช่น เบอร์โทรศัพท์ผิด, โลโก้ขาดหาย, ที่อยู่ไม่ถูก หรือสีโลโก้เพี้ยนไปจากมาตรฐานแบรนด์ ทั้งหมดนี้รู้เมื่อรับงานไปแล้ว และต้องพิมพ์ใหม่ทั้งหมด

**Proof** หรือการขอดูตัวอย่างก่อนพิมพ์จริง คือขั้นตอนที่คุ้มค่าที่สุดในกระบวนการพิมพ์ ไม่ว่าจะเป็น Soft Proof (ดูบนหน้าจอ) หรือ Hard Proof (พิมพ์ตัวอย่าง 1 ชิ้น) ก็ตาม

**วิธีป้องกัน:** ก่อนอนุมัติพิมพ์ทุกครั้ง ให้เช็กลิสต์นี้

- ข้อความถูกต้องครบถ้วนทุกตัวอักษรหรือไม่
- เบอร์โทร, อีเมล, เว็บไซต์ ถูกต้องไหม
- สีโลโก้ตรงกับ Brand Guide ไหม
- ขนาดและทิศทางของงานถูกต้องไหม

ให้คนที่ไม่ได้ทำงานนี้ช่วยอ่านทบทวนด้วย เพราะตาเราเองมักมองข้ามสิ่งที่คุ้นเคย
