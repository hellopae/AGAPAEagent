# PART 1 — บุญ Quest (Deep Research)

## 1. หลักฐานรายได้: print-and-play / hybrid digital+physical ขายได้จริงไหม?

### 1.1 ฝั่งดิจิทัล/PnP บน itch.io

- ✅ VERIFIED — เกมส่วนใหญ่บน itch.io ทำรายได้**ต่ำกว่า $100 ตลอดอายุเกม** จากเกมกว่า 900,000 เกมบนแพลตฟอร์ม เกมที่ทำการตลาดจริงจังมักอยู่ช่วง $500–5,000 ([generalistprogrammer.com — Itch.io Revenue Guide 2026](https://generalistprogrammer.com/tutorials/how-to-make-money-on-itchio-indie-game-guide)) — *หมายเหตุความน่าเชื่อถือ: เป็น guide site ไม่ใช่ข้อมูลทางการจาก itch.io แต่สอดคล้องกับกระทู้ community ของ itch.io เอง*
- ✅ VERIFIED — DriveThruCards (print-on-demand การ์ดเกม): ผู้สร้างที่ขาย exclusive ได้ **70% ของ margin ~$12/สำรับ = ~$8.40/สำรับที่ขายได้** ([DriveThru Partners — Card Printing FAQ](https://help.drivethrupartners.com/hc/en-us/articles/12780818417687)) — ยืนยันว่าโมเดล "การ์ดเกม margin สูงต่อชิ้น" มีจริง แต่ไม่มีข้อมูล volume
- ✅ VERIFIED — เคสบวกสุดของ PnP: Matthew Dunstan (ผู้ร่วมก่อตั้ง Postmark Games — สำนักพิมพ์ print-at-home) รายงานรายได้จากงานออกแบบเกมปี 2023 = **€164,423** ([Finances of a Game Designer 2023](https://matthewdunstan.substack.com/p/finances-of-a-game-designer-2023)) — *ข้อควรระวัง: เป็น outlier ระดับท็อป มีหลายรายได้รวมกัน ไม่ใช่ median ของตลาด*
- ⚠️ UNVERIFIED — ยอดขาย median เฉพาะหมวด "deckbuilder / print-and-play card game" บน itch.io — ไม่มีข้อมูลสาธารณะแยกหมวด หา postmortem ที่เปิดตัวเลขตรง ๆ ไม่พบในรอบค้นนี้
- 💬 OPINION — สรุปเชิงวิเคราะห์: PnP/ดิจิทัลอย่างเดียว "เลี้ยงตัวไม่ได้" ในเคสทั่วไป ตัวเลขที่สมเหตุสมผลสำหรับเกมแรกคือหลักร้อย–หลักพัน USD ต่อปี ถ้าทำการตลาดดี ดังนั้นสมมติฐานของ Minnie ที่ให้ digital เป็น "demo/marketing" ไม่ใช่แหล่งรายได้หลัก **ถูกทิศทางแล้ว**

### 1.2 Crowdfunding / เกมโต๊ะระดับ indie

- ✅ VERIFIED — Kickstarter หมวด tabletop ระดมทุนรวม ~$220M ในปี 2024 และเกมแนว solo/co-op เป็น segment ที่โตต่อเนื่อง ([BoardGameWire](https://boardgamewire.com/index.php/2025/02/06/our-ambition-for-2025-is-to-be-number-one-in-tabletop-gamefound-closes-gap-on-kickstarter-again-as-crowdfunding-giants-2024-dollars-raised-remains-flat/))
- ⚠️ UNVERIFIED — ค่าเฉลี่ย/median ของแคมเปญการ์ดเกมเล็ก ๆ (solo dev, cultural theme) — ไม่พบข้อมูลรวมที่เชื่อถือได้
- ⚠️ UNVERIFIED — เคส Kickstarter การ์ดเกมธีมผี/วัฒนธรรมไทยโดยเฉพาะ — ค้นไม่พบแคมเปญที่โดดเด่น (มีเคสญี่ปุ่น เช่น yokai card games จำนวนมาก แต่ไทยแทบไม่มีบน KS) — 💬 นี่เป็นทั้ง "ช่องว่าง" และ "สัญญาณเตือน" ว่ายังไม่มีใครพิสูจน์ demand เส้นทางนี้

### 1.3 ตลาดบอร์ดเกมไทย (จุดแข็งสุดของไอเดียนี้)

- ✅ VERIFIED — มูลค่าตลาดบอร์ดเกมไทย: **2,329 ล้านบาท (2024) → คาดการณ์ 2,562 ลบ. (2025) → 2,795 ลบ. (2026) → ~3,028 ลบ. (2027)** ([Marketeer Online](https://marketeeronline.co/archives/448867))
- ✅ VERIFIED — ประมาณการอีกสำนัก: ตลาดบอร์ดเกมไทย **USD 74.12M (2024) → USD 198.09M (2033), CAGR 10.33%** ([IMARC Group](https://www.imarcgroup.com/thailand-board-games-market))
- ✅ VERIFIED — สัดส่วน**เกมฝีมือคนไทย**เพิ่มจาก ~5–10% เป็น **~20% ของตลาด** ([Marketeer Online](https://marketeeronline.co/archives/448867))
- ✅ VERIFIED — ช่องทางขายปี 2024: ร้านค้าปลีก/ห้าง/ร้านหนังสือ ~50%, ร้านเฉพาะทาง ~25%, ออนไลน์+อื่น ๆ ~25% ([Marketeer Online](https://marketeeronline.co/archives/448867))
- ✅ VERIFIED — มีร้าน/คาเฟ่บอร์ดเกม **~150 แห่งทั่วไทย** ส่วนใหญ่ใน กทม. และมีสตูดิโอออกแบบเกมสัญชาติไทย **40+ แห่ง** งานเกมจัดแทบทุกเดือน ([Wizards of Learning](https://wizardsoflearning.com/overview-boardgames-in-thailand-2019/), [readthecloud.co](https://readthecloud.co/10-thai-boardgame/), [Thai PBS](https://www.thaipbs.or.th/news/content/321158))
- ✅ VERIFIED — ผู้เล่นหลักในตลาด: สยามบอร์ดเกม, Tower Tactic Games (นำเข้า+แปลไทย), Lanlalen, Wizards of Learning, Thailand Board Game Association ([thailandboardgame.org](https://thailandboardgame.org/), [lanlalen.com](https://www.lanlalen.com/en))
- ✅ VERIFIED — มีเกมไทยธีมความเชื่อ/ศาสนาอยู่แล้ว เช่นบอร์ดเกม **"ผี พระ เจ้า"** (สื่อสารเรื่องศาสนาและความหลากหลายทางวัฒนธรรม) และโปรเจกต์ **Thai Board Game Journey 2025 ธีม "ผีอาเซียน"** ของ TK Park ([Mutual Finding](https://mutualfinding.co/board-game/), [TK Park](https://mytk.tkpark.or.th/Activity/ActivityInformation?id=536&mode=website)) — 💬 แปลว่าธีมพุทธ/ความเชื่อในบอร์ดเกม "ผ่านการยอมรับทางสังคมมาแล้ว" ระดับหนึ่ง ถ้าทำอย่างเคารพ
- ✅ VERIFIED — เกมดิจิทัลธีมไทยขายตลาดโลกได้จริง: **Home Sweet Home** (Yggdrazil, ผีไทย) แฟรนไชส์ขายเกิน **1 ล้านชุด**, รายได้ปี 2018 = 55.6 ล้านบาท ([Wikipedia + MGR Online/Brand Inside อ้างใน Wikipedia](https://en.wikipedia.org/wiki/Home_Sweet_Home_(2017_video_game))) — พิสูจน์ว่า "วัฒนธรรมไทย + เกม" มี demand ระดับโลก (แม้จะเป็น horror ไม่ใช่ deckbuilder)

## 2. Cultural / Sensitivity Check — เส้นแดงธีมพุทธ (สำคัญที่สุดของ brief นี้)

### 2.1 Precedent ที่เป็นบทเรียน

- ✅ VERIFIED — เกม **"Fight of Gods"** (พระพุทธเจ้าต่อยกับพระเยซู) ถูก**แบนในไทยปี 2017** — กระทรวงดีอีบล็อก Steam ชั่วคราวจนกว่าจะเอาเกมออกจากการขายในไทย หลังองค์กรพุทธร้องเรียน ([Khaosod English](https://www.khaosodenglish.com/life/arts/2017/09/14/thailand-bans-fight-gods-game/), [The Nation](https://www.nationthailand.com/life/30326833))
- ✅ VERIFIED — **Knowing Buddha Organization (KBO)** องค์กรไทยก่อตั้ง 2012 รณรงค์เชิงรุกต่อการใช้พระพุทธรูป/ภาพพระพุทธเจ้าเชิงพาณิชย์ — เคยกดดันสำเร็จหลายเคสทั้งในและต่างประเทศ (ฝาชักโครกลายพระในฝรั่งเศส/เนเธอร์แลนด์, ภาพในนิตยสาร ฯลฯ) หลักการของ KBO: "พระพุทธรูปควรอยู่ในที่บูชา ไม่ใช่ของตกแต่ง/สินค้า" ([knowingbuddha.org](https://knowingbuddha.org/), [Wikipedia](https://en.wikipedia.org/wiki/Knowing_Buddha), [Buddhistdoor](https://www.buddhistdoor.net/features/the-buddha-image-branding-commercialization-denigration/))

### 2.2 เส้นแบ่ง "ทำได้ / ห้ามทำ" (สังเคราะห์จากหลักฐานข้างบน)

💬 OPINION (อิงหลักฐาน precedent ✅ ข้างต้น) — ข้อเสนอ guardrails สำหรับ บุญ Quest:

**ห้ามเด็ดขาด (hard NO):**
1. **ห้ามมีรูปพระพุทธเจ้า/พระพุทธรูปบนการ์ด** ทุกกรณี — นี่คือเส้นที่ Fight of Gods และเคส KBO ทั้งหมดชี้ตรงกัน ยิ่งเป็นสินค้า "ซื้อขาย จับ สับ กอง วางบนโต๊ะ ทำหล่น" ยิ่งเสี่ยง (การ์ดถูกวางต่ำกว่าเอว/ทับกัน = ผิดมารยาททางศาสนาในสายตาชาวพุทธเคร่ง)
2. ห้ามให้พระพุทธเจ้า/พระอรหันต์เป็น "ตัวละครที่สู้กัน" หรือถูกโจมตี
3. ห้ามใช้พระสงฆ์จริง/สิ่งศักดิ์สิทธิ์เฉพาะ (เช่น พระแก้วมรกต) เป็น game piece
4. ห้ามผูกกลไก "จ่ายเงินจริงเพื่อได้บุญ" — จะโดนวิจารณ์เรื่องพุทธพาณิชย์ทันที

**ทำได้ (โซนปลอดภัยที่มี precedent รองรับ):**
- ธีม **การสั่งสมบารมี/ต่อสู้กิเลสมาร** ในเชิงนามธรรม (กิเลส โลภ โกรธ หลง เป็นมอนสเตอร์เชิงสัญลักษณ์ — มีรากในพุทธศิลป์เรื่อง "มารผจญ" อยู่แล้ว)
- โฟล์คลอร์ไทย ผี ยักษ์ เทวดา สัตว์หิมพานต์ — เกมไทย ("ผี พระ เจ้า", Home Sweet Home, งานผีอาเซียนของ TK Park) ใช้แล้วไม่มีดราม่า
- ฉากวัด ลายไทย สถาปัตยกรรม ระฆัง เจดีย์ (เป็น setting ไม่ใช่วัตถุบูชา)
- ตัวละคร "ผู้บำเพ็ญ/ผู้เดินทางสายบุญ" ที่เป็นคนธรรมดา (ไม่ใช่พระสงฆ์ทำผิดวินัย)

**ควรทำเพิ่ม:** 💬 ให้ Chris QA + คนไทยสายพุทธเคร่ง 2–3 คน review อาร์ตทุกใบก่อนพิมพ์ และเขียนหมายเหตุความเคารพ (respect note) ในกล่อง/หน้าเกม

## 3. Distribution Reality — ขายการ์ดเกมจริงในไทยแบบ solo

- ✅ VERIFIED — ราคาเกมภาษาไทยในตลาดจริง (Tower Tactic Games, ก.ค. 2026): การ์ดเกมเล็กเริ่ม **549–890฿** (Yogi 555฿, TRIO 550฿, Taco Cat 549฿, Insider 890฿), เกมกล่องกลาง-ใหญ่ **1,190–1,599฿** ส่วนใหญ่กระจุก **800–1,600฿** ([towertacticgames.com](https://www.towertacticgames.com/en/collections/thai-version))
  - 💬 ราคาที่ Minnie ตั้ง (390–590฿) อยู่**ขอบล่างของตลาด** — แข่งขันได้และมีที่ว่าง เพราะ TANAPAT พิมพ์เองไม่มีคนกลาง แต่ถ้าคุณภาพการ์ด+กล่องดี ขึ้นไป 590–790฿ ก็ยังอยู่ในช่วงที่ตลาดรับ
- ✅ VERIFIED — งานอีเวนต์หลัก: **Thailand Board Game Show 2025** จัดครั้งที่ 6 ที่ Union Mall (7–9 พ.ย. 2025) มี **50+ บูธ** นักออกแบบไทย+ต่างชาติ ขยายพื้นที่จาก 2 เป็น 3 โซน ([thailandboardgameshow.com](https://www.thailandboardgameshow.com/), [BGG — Hiew's Blog](https://boardgamegeek.com/blog/27/blogpost/180085/thailand-board-game-show-2025-quick-takes-part-1)) — 💬 นี่คือช่องเปิดตัวที่เหมาะที่สุดสำหรับ first print run (พ.ย. 2026)
- ✅ VERIFIED — ช่องทางขายในไทยกระจาย: ค้าปลีก 50% / ร้านเฉพาะทาง 25% / ออนไลน์ 25% และ Shopee มีหมวด "บอร์ดเกมไทย" ค้นหาได้เป็นเรื่องปกติ ([Marketeer](https://marketeeronline.co/archives/448867), [Shopee TH](https://shopee.co.th/search?keyword=บอร์ดเกมไทย))
- ⚠️ UNVERIFIED — ค่าขนส่งการ์ดเกมในไทย/ต่างประเทศแบบตัวเลขแน่นอน — ไม่ได้ตรวจรอบนี้ (💬 ประมาณการทั่วไป: ในไทยผ่าน Flash/J&T หลักสิบบาทต่อกล่องเล็ก ไม่กินกำไรมาก; ส่งออกต่างประเทศคือจุดที่กินกำไรจริง มัก 250–600฿+/ชิ้นสำหรับ tracked parcel — **ต้องตรวจจริงกับไปรษณีย์ไทย/DHL ก่อนเปิดขาย worldwide** ห้ามใช้ตัวเลขนี้ตัดสินใจ)
- 💬 OPINION — เส้นทางขายที่เหมาะกับ TANAPAT: (1) LINE OA + PromptPay สำหรับแฟนตรง (margin สูงสุด) → (2) Shopee เพื่อ discovery → (3) ฝากขาย/demo ตามคาเฟ่บอร์ดเกม 5–10 แห่งใน กทม. → (4) บูธ Thailand Board Game Show ปลายปี ส่วนขาย worldwide ให้เริ่มจาก PnP PDF (Gumroad/itch.io) ก่อน อย่าเพิ่งแบกค่าส่งกายภาพข้ามประเทศ

## 4. ฝั่งดิจิทัล — demand ของ web deckbuilder

- ✅ VERIFIED — itch.io มี tag deckbuilder / slay-the-spire ที่คึกคัก มีเกม web-playable จำนวนมาก ([itch.io deckbuilder web tag](https://itch.io/games/platform-web/tag-deckbuilder), [slay-the-spire tag](https://itch.io/games/tag-slay-the-spire)) — ยืนยันว่า audience หมวดนี้มีอยู่และเล่นบนเบราว์เซอร์จริง
- ⚠️ UNVERIFIED — ตัวเลข download/รายได้ median ของ web deckbuilder บน itch.io — ไม่มีข้อมูลสาธารณะ; postmortem แบบเปิดตัวเลขของหมวดนี้หาไม่พบในรอบค้น
- 💬 OPINION — สำหรับ บุญ Quest ให้ตั้งเป้าฝั่งดิจิทัลเป็น **KPI แบบ funnel ไม่ใช่รายได้**: จำนวนคนเล่นจบ 1 run, จำนวนคลิกไปหน้า pre-order การ์ดจริง, conversion ≥2% ตามสมมติฐานของ Minnie คือสิ่งที่ต้องวัด ไม่ใช่ยอดขาย pay-what-you-want

## 5. VERDICT — บุญ Quest: **GO (แบบมีเงื่อนไข)**

💬 OPINION (สังเคราะห์จากหลักฐานทั้งหมดข้างบน):

**เหตุผลฝั่ง GO:**
1. ตลาดบอร์ดเกมไทยโตจริง (✅ ~2.3–2.8 พันล้านบาท, CAGR ~10%) และสัดส่วนเกมไทยเพิ่มเป็น ~20% — ลมหนุนชัดเจน

4. มี ecosystem รองรับ: คาเฟ่ ~150 แห่ง, งานโชว์ประจำปี 50+ บูธ, สมาคมบอร์ดเกมไทย

**เงื่อนไขบังคับ (ถ้าไม่ทำ = NO-GO):**
1. **Art guardrails ข้อ 2.2 ต้องถือเป็นกฎเหล็ก** — โดยเฉพาะ "ไม่มีพระพุทธรูป/พระพุทธเจ้าบนการ์ด" (ความเสี่ยงไม่ใช่แค่ดราม่า แต่คือแบรนด์โรงพิมพ์ 40 ปีของครอบครัว)
2. **พิสูจน์ demand ก่อนพิมพ์**: ลำดับ = digital demo ฟรี → ขาย PnP PDF → เปิด pre-order การ์ดจริงผ่าน LINE → พิมพ์เมื่อยอด pre-order ถึงจุดคุ้มทุน batch แรก (small batch ที่โรงพิมพ์ตัวเองทำได้คือข้อได้เปรียบที่คู่แข่งไม่มี)
3. **คาดหวังรายได้ดิจิทัลต่ำ**: วางแผนการเงินโดยสมมติ digital ≈ 0 บาท เป็น marketing cost ที่จ่ายด้วยแรง agent
4. อย่าเพิ่งทำ worldwide physical จนกว่าจะตรวจค่าส่งจริง (⚠️ ยังไม่ verified)

**ความเสี่ยงเปิดที่ใหญ่สุด:** ไม่มี comparable ตรง ๆ ของ "การ์ดเกมไทย indie ขายผ่าน LINE" ที่เปิดเผยยอดขาย — เรากำลังเป็นคนพิสูจน์เส้นทางนี้เอง จึงต้องออกแบบให้ล้มได้ถูก (fail cheap) ผ่าน pre-order gate

- 💬 OPINION — Guardrails ที่ต้องมี: (1) ระบุชัดว่าเป็น **"เกม/ความบันเทิง"** ไม่ใช่บริการพยากรณ์จริง (2) **ห้ามขายพิธีกรรม/แก้กรรม/วัตถุมงคลอวดอ้างผล** — ขายได้แค่ "เนื้อหาเกมเพิ่ม" (3) ไม่เก็บข้อมูลส่วนตัวอ่อนไหวเกินจำเป็น (PDPA) (4) มี disclaimer ก่อนเริ่มเล่น — ทำ 4 ข้อนี้แล้วความเสี่ยงกฎหมายต่ำ

## 3. ต้นทุน Claude API vs ราคาขาย

- ✅ VERIFIED — ราคา API ปัจจุบัน (จาก reference ทางการของ Anthropic, มิ.ย. 2026): **Haiku 4.5 = $1 input / $5 output ต่อ 1M tokens**, Sonnet 5 = $3/$15 (โปรโมชัน intro $2/$10 ถึง 31 ส.ค. 2026), Opus 4.8 = $5/$25
- 💬 OPINION — ประมาณการต้นทุนต่อเซสชัน 15 นาที (~15 calls, ประวัติสนทนาสะสม, เฉลี่ย input ~4K tokens/call, output ~300 tokens/call):
  - **Haiku 4.5:** input ~60K × $1/M + output ~4.5K × $5/M ≈ **$0.08/เซสชัน (~3 บาท)** — ถ้าใช้ prompt caching (ระบบ persona+ประวัติ cache ได้) ลดเหลือ **~1–1.5 บาท/เซสชัน**
  - **Sonnet 5 (intro):** ≈ $0.14–0.16/เซสชัน (~5–6 บาท) — ภาษาไทยสวยกว่า เหมาะกับบท "แม่หมอ"
  - เทียบราคาขายแพ็ค 59฿: ครอบคลุมต้นทุน Haiku ได้ ~20–40 เซสชัน หรือ Sonnet ~10 เซสชัน → **margin ≥70% ตามที่ Minnie ตั้งไว้ "เป็นไปได้"** ถ้าจำกัดจำนวนเซสชัน/แพ็ค และฟรีเทียร์จำกัด 1 ครั้ง/วันด้วย Haiku
  - ⚠️ UNVERIFIED — ตัวเลข tokens ต่อเซสชันเป็นการประเมิน ต้อง prototype วัดจริงก่อนตั้งราคา

## 4. VERDICT — หมอดูแม่นจริง: **GO แบบระวัง (conditional GO)**

💬 OPINION — ตลาดจ่ายจริง (✅ ธุรกิจมูโต +70–113%, แพลตฟอร์มจ่ายเงินมีอยู่แล้ว), ต้นทุน API คุมได้ (~1–6 บาท/เซสชัน), กฎหมายผ่านได้ถ้าวางตัวเป็น "เกมบันเทิง" ไม่อวดอ้าง — ความเสี่ยงหลักคือ (1) คู่แข่ง ChatGPT ฟรี → ต้องชนะด้วยตัวละคร+เนื้อเรื่อง+อาร์ต ไม่ใช่ความแม่น (2) ต้องมี backend/การจัดการ API key ซึ่งเพิ่ม complexity กว่า บุญ Quest — จัดเป็นไอเดียลำดับ 2 ตามที่ Minnie จัดไว้ เหมาะทำหลัง บุญ Quest มี playbook แล้ว และ synergy "ไพ่ยิปซีลายไทยพิมพ์จริง" ใช้ funnel เดียวกับ บุญ Quest ได้เลย
