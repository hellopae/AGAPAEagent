## 1) Feature Concept (1 หน้า)

### ปัญหา

คนที่ต้องออกเกียรติบัตร "เป็นชุด" — ครูจัดแข่งขัน 80 คน, HR อบรมพนักงาน 45 คน, วัดขอบคุณผู้บริจาค 120 ราย — วันนี้ทำแบบนี้:

1. มีรายชื่ออยู่แล้วในไฟล์ Word/Excel/PDF (ใบสมัคร, ทะเบียนอบรม, บัญชีผู้บริจาค)
2. เปิด Word หรือ Canva → พิมพ์/วางชื่อทีละคน → เซฟทีละใบ
3. ชื่อยาวล้นกรอบ, ฟอนต์ไทยเพี้ยน, สะกดผิดเพราะพิมพ์ซ้ำด้วยมือ
4. เสียเวลาครึ่งวันถึงทั้งวัน กับงานที่เครื่องควรทำให้ใน 1 นาที

เครื่องมือที่มีอยู่ไม่ตอบโจทย์คนไทยทั่วไป: autoCrat ต้องตั้งค่า Google Sheets + mail merge ซึ่งครูจำนวนมากไม่ถนัด, Canva bulk create อยู่ใน tier จ่ายเงินและ template ส่วนใหญ่เป็นสไตล์ฝรั่ง ไม่เหมือน "เกียรติบัตรราชการ/สถาบันไทย" ที่หน่วยงานคาดหวัง *(สมมติฐาน — Reese ตรวจ)*

### Solution

**เกียรติบัตร Express** — เว็บแอปหน้าเดียว ใช้ได้ทันทีไม่ต้องสมัครสมาชิก:

> **ลากไฟล์รายชื่อมาวาง (Word/PDF/Excel) → ระบบดึงรายชื่อให้ → เลือก template เกียรติบัตรสไตล์ไทย → เห็น preview จริงทุกใบ → ดาวน์โหลด PDF ทั้งชุด พร้อมพิมพ์**

จุดต่างสำคัญ 3 ข้อ:

1. **"เอาไฟล์ที่มีอยู่แล้วมาใช้ได้เลย"** — ไม่บังคับกรอกฟอร์มใหม่ ไม่บังคับจัด format ก่อน ระบบอ่าน Word/PDF/Excel แล้วเดารายชื่อให้ ผู้ใช้แค่ตรวจ/แก้ในตาราง
2. **Template "ของจริงแบบไทย"** — ดีไซน์จากประสบการณ์พิมพ์เกียรติบัตรให้สถาบันไทย 40 ปีของ TANAPAT: ลายกรอบ, ตราประทับ, ลำดับข้อความ, ฟอนต์ไทยที่ถูกกาลเทศะ — ฟรี 3 แบบ, แบบพรีเมียมโชว์ preview ให้เห็นก่อนแล้วติดต่อซื้อทาง LINE/email
3. **ทำงานบนเครื่องผู้ใช้ทั้งหมด** — ไฟล์รายชื่อไม่ถูกอัปโหลดขึ้น server (เพราะไม่มี server) → ขายเป็นจุดแข็งเรื่องความเป็นส่วนตัวของข้อมูลนักเรียน/พนักงาน/ผู้บริจาคได้เลย

### ทำไม TANAPAT ชนะ

| ข้อได้เปรียบ | คู่แข่งลอกยากเพราะ |
|---|---|
| Template ดู "เป็นทางการแบบไทยแท้" | 40 ปีของงานพิมพ์เกียรติบัตรสถาบัน — รู้ว่ากรอบ/ตรา/ถ้อยคำแบบไหน "ผ่าน" ในสายตาผู้อำนวยการโรงเรียนหรือเจ้าอาวาส ไม่ใช่ clipart ฝรั่ง |
| Upsell พิมพ์จริง | ปลายทางของเกียรติบัตรคือ "กระดาษ" — TANAPAT พิมพ์บนกระดาษอาร์ต + ปั๊มทอง ส่งถึงที่ได้เอง คู่แข่ง software-only ไม่มีโรงพิมพ์ |
| ต้นทุนรันเกือบศูนย์ | Client-side ล้วนบน GitHub Pages — ไม่มีค่า server ต่อผู้ใช้ ปล่อยให้คนใช้ฟรีจำนวนมากได้โดยไม่เจ็บ |
| Pipeline ผลิต template ด้วยทีม AI | ออก template พรีเมียมใหม่ตามฤดูกาล (วันเด็ก, กีฬาสี, งานวิชาการ, กฐิน) ได้เร็วและถูกกว่าคู่แข่งที่ต้องจ้างดีไซเนอร์ |

**เส้นทางรายได้ MVP:** template พรีเมียม (ติดต่อซื้อผ่าน LINE/email — ปิดการขายด้วยมือ) → ต่อยอดเป็น upsell งานพิมพ์จริง ซึ่ง margin สูงกว่าและเป็นธุรกิจหลักของ TANAPAT อยู่แล้ว การขายผ่าน LINE ใน MVP ไม่ใช่ข้อด้อย — มันคือช่องคุยที่เปิดโอกาสเสนองานพิมพ์จริงทันที

## 2) User Stories

### Persona A — ครู (เช่น ครูฝ่ายวิชาการ/กิจกรรม โรงเรียนประถม-มัธยม)

- ในฐานะครูที่จัดการแข่งขันวิชาการ ฉันอยากจะอัปโหลดไฟล์ Excel รายชื่อนักเรียนที่มีอยู่แล้ว เพื่อไม่ต้องพิมพ์ชื่อนักเรียน 80 คนซ้ำอีกรอบ (และไม่พิมพ์สะกดผิดจนผู้ปกครองโทรมา)
- ในฐานะครู ฉันอยากจะเลือก template ที่ดูเป็นทางการแบบเกียรติบัตรโรงเรียนไทยจริง ๆ เพื่อให้ผู้อำนวยการอนุมัติโดยไม่สั่งแก้
- ในฐานะครู ฉันอยากจะแก้ไขข้อความกลาง (ชื่องาน, "ได้เข้าร่วม/ได้รับรางวัล...", วันที่, ชื่อผู้ลงนาม) ได้เอง เพื่อใช้ template เดียวกับหลายงานตลอดปี
- ในฐานะครู ฉันอยากจะเห็น preview เกียรติบัตรของนักเรียนที่ชื่อยาวที่สุด เพื่อมั่นใจว่าชื่อไม่ล้นกรอบก่อนกดพิมพ์ทั้งชุด
- ในฐานะครู ฉันอยากจะดาวน์โหลด PDF ทั้งชุดในไฟล์เดียว เพื่อส่งเข้าเครื่องถ่ายเอกสารของโรงเรียนได้ทันที
- ในฐานะครูที่ไม่เก่งคอมพิวเตอร์ ฉันอยากจะทำทั้งหมดนี้ได้โดยไม่ต้องสมัครสมาชิกหรือตั้งค่าอะไร เพื่อให้เสร็จในคาบว่างคาบเดียว

### Persona B — HR (ฝ่ายบุคคล/ฝ่ายฝึกอบรมบริษัท)

- ในฐานะ HR ที่จัดอบรมพนักงาน ฉันอยากจะอัปโหลดทะเบียนผู้เข้าอบรม (Excel ที่ export จากระบบลงทะเบียน) เพื่อออกวุฒิบัตรให้ผู้ผ่านการอบรมทุกคนภายในวันเดียวกับที่อบรมจบ
- ในฐานะ HR ฉันอยากจะใส่โลโก้บริษัทลงใน template เพื่อให้วุฒิบัตรดูเป็นเอกสารทางการขององค์กร
- ในฐานะ HR ฉันอยากจะมั่นใจว่ารายชื่อพนักงานไม่ถูกส่งขึ้น server ของใคร เพื่อไม่ผิดนโยบายข้อมูลส่วนบุคคล (PDPA) ของบริษัท
- ในฐานะ HR ฉันอยากจะได้ทั้ง PDF รวมชุด (ไว้พิมพ์) และเข้าใจว่าจะได้ไฟล์แยกต่อคนได้อย่างไร เพื่อแนบวุฒิบัตรเข้าอีเมลรายบุคคลหรือแฟ้มพนักงาน
- ในฐานะ HR ที่เจอ template พรีเมียมสวยถูกใจ ฉันอยากจะติดต่อซื้อผ่านช่องทางที่เป็นลายลักษณ์อักษร (email) เพื่อขอใบเสร็จ/เอกสารไปเบิกกับบริษัทได้

### Persona C — วัด / มูลนิธิ (เจ้าหน้าที่ฝ่ายธุรการ, กรรมการวัด, อาสาสมัคร)

- ในฐานะเจ้าหน้าที่มูลนิธิ ฉันอยากจะเอาบัญชีรายชื่อผู้บริจาค (มักเป็น Word หรือ PDF ที่พิมพ์ต่อ ๆ กันมา) มาให้ระบบดึงชื่อ เพื่อออกใบอนุโมทนา/เกียรติบัตรขอบคุณผู้บริจาคทั้งร้อยรายโดยไม่ต้องจ้างร้านพิมพ์ทำ artwork
- ในฐานะกรรมการวัด ฉันอยากจะได้ template โทนพุทธ-ไทยที่สีและถ้อยคำถูกธรรมเนียม เพื่อให้เหมาะกับงานบุญและไม่ผิดกาลเทศะ
- ในฐานะอาสาสมัครที่ไม่ชำนาญเทคโนโลยี ฉันอยากจะทำตามขั้นตอนที่มีไม่กี่ปุ่มและเป็นภาษาไทยทั้งหมด เพื่อทำเองได้โดยไม่ต้องรอลูกหลานมาช่วย
- ในฐานะเจ้าหน้าที่วัดที่จะแจกเกียรติบัตรในงานกฐิน ฉันอยากจะสั่งพิมพ์จริงบนกระดาษอย่างดี-ปั๊มทองจากผู้ให้บริการเดียวกันได้เลย เพื่อให้ของที่มอบมือผู้บริจาคดูสมเกียรติ *(นี่คือประตูสู่ upsell งานพิมพ์ TANAPAT)*

## 3) User Flow หลักของ MVP

```
[หน้าแรก] → 1. Upload → 2. Map ชื่อ → 3. เลือก template → 4. ปรับข้อความ → 5. Preview → 6. Export PDF
```

**ขั้น 0 — หน้าแรก**
เห็นตัวอย่างเกียรติบัตรสำเร็จ + ปุ่มเดียว "เริ่มทำเกียรติบัตร" + ข้อความยืนยัน "ไฟล์ของคุณไม่ถูกอัปโหลดไปที่ไหน ทุกอย่างทำบนเครื่องคุณ" ไม่มี login

**ขั้น 1 — Upload รายชื่อ**
- ลากวางหรือเลือกไฟล์ .docx / .xlsx / .pdf (อ่านด้วย lib ฝั่ง client — Reese กำลังหาตัวที่ใช่)
- ทางเลือกสำรอง (สำคัญมาก): **วางรายชื่อจาก clipboard** ลงกล่องข้อความ — เป็น fallback เมื่อ parser อ่านไฟล์ไม่สำเร็จ ทำให้ผู้ใช้ไม่จนมุม

**ขั้น 2 — Map ชื่อ (ตรวจ/แก้รายชื่อ)**
- ระบบเดาว่าคอลัมน์/บรรทัดไหนคือ "ชื่อ-นามสกุล" แล้วแสดงเป็นตารางแก้ไขได้
- ผู้ใช้ลบแถวที่ไม่ใช่ชื่อ (หัวตาราง, เลขลำดับ), แก้คำผิด, เพิ่มชื่อที่ตกหล่น
- แสดงจำนวนรวม: "พบ 47 รายชื่อ" — ตัวเลขนี้คือ contract ก่อน generate
- (ถ้าไฟล์มีคอลัมน์อื่น เช่น รางวัล/หน่วยงาน — MVP เก็บเฉพาะคอลัมน์ชื่อ ช่องอื่นรอเฟสหน้า)

**ขั้น 3 — เลือก template**
- แกลเลอรี: ฟรี 3 แบบ (ทางการ-ราชการ / พุทธ-ไทย / โมเดิร์น-องค์กร) + พรีเมียมอีกจำนวนหนึ่ง
- พรีเมียม: เห็น preview เต็มตา (มีลายน้ำ/ล็อกดาวน์โหลด) + ปุ่ม "ติดต่อซื้อแบบนี้" → เปิด LINE OA หรือ mailto พร้อมข้อความอัตโนมัติระบุรหัส template

**ขั้น 4 — ปรับข้อความกลาง**
- ฟอร์มกรอก: ชื่อหน่วยงาน/โรงเรียน, ชื่องาน/หลักสูตร, ข้อความรับรอง ("ขอมอบเกียรติบัตรฉบับนี้เพื่อแสดงว่า..."), วันที่ (พ.ศ.), ชื่อ-ตำแหน่งผู้ลงนาม 1–2 คน
- อัปโหลดโลโก้หน่วยงาน (PNG) วางในตำแหน่งที่ template กำหนด

**ขั้น 5 — Preview**
- เห็นเกียรติบัตรจริงทีละใบ เลื่อนดูได้ทุกชื่อ
- ระบบชี้เตือน "ชื่อยาวเสี่ยงล้นกรอบ" พร้อมย่อฟอนต์อัตโนมัติเฉพาะใบนั้น (auto-fit)
- แก้อะไรก็ย้อนกลับขั้น 2–4 ได้โดยรายชื่อไม่หาย

**ขั้น 6 — Export PDF**
- ปุ่มเดียว "ดาวน์โหลด PDF ทั้งชุด (47 ใบ)" → ไฟล์ PDF รวม 1 ไฟล์ ขนาด A4 แนวนอน 1 ใบ/หน้า ฝังฟอนต์ไทย พร้อมพิมพ์
- หน้าจบงานแสดง upsell: "อยากได้แบบพิมพ์จริง กระดาษอาร์ต ปั๊มทอง ส่งถึงที่? ทัก LINE TANAPAT"

## 4) ขอบเขต MVP — IN / OUT

### IN (เวอร์ชันแรก)

| # | ฟีเจอร์ | หมายเหตุ |
|---|---|---|
| 1 | Upload .docx / .xlsx / .pdf + ดึงรายชื่อฝั่ง client | pdf อาจแม่นต่ำสุด — ต้องมี fallback |
| 2 | วางรายชื่อจาก clipboard (fallback) | กันผู้ใช้จนมุมเมื่อ parser พลาด |
| 3 | ตารางตรวจ/แก้/ลบ/เพิ่มรายชื่อ | คอลัมน์เดียว: ชื่อ-นามสกุล |
| 4 | Template ฟรี 3 แบบ (ทางการ / พุทธ-ไทย / องค์กร) | ดีไซน์โดย Vera+Mind ตาม SOP-10 |
| 5 | Template พรีเมียม: preview + ลายน้ำ + ล็อก + ปุ่มติดต่อซื้อ LINE/email | ไม่มีระบบจ่ายเงิน — ปิดการขายด้วยมือ |
| 6 | แก้ข้อความกลาง: หน่วยงาน, ชื่องาน, ข้อความรับรอง, วันที่ พ.ศ., ผู้ลงนาม 1–2 คน | |
| 7 | อัปโหลดโลโก้หน่วยงาน (PNG/JPG) | วางตำแหน่งตายตัวตาม template |
| 8 | Preview ทุกใบ + auto-fit ชื่อยาว | จุด quality ที่คู่แข่งมักพลาด |
| 9 | Export PDF รวมชุด, A4 แนวนอน, ฝังฟอนต์ไทย | lib ตามผล research ของ Reese |
| 10 | หน้า upsell พิมพ์จริง (ลิงก์ LINE OA) | ประตูสู่รายได้งานพิมพ์ |
| 11 | Deploy GitHub Pages + .nojekyll | ตาม skill `deploy-pages` |

### OUT (รอเฟสหน้า — จดไว้ ไม่ทำตอนนี้)

| # | ฟีเจอร์ | เหตุผลที่รอ |
|---|---|---|
| 1 | ระบบจ่ายเงินในเว็บ (PromptPay/บัตร) | ล็อกโดย Kittanate — MVP ขายผ่าน LINE/email |
| 2 | ระบบสมาชิก / login / เก็บงานเก่า | client-side ล้วน ไม่มี backend; เพิ่ม friction โดยไม่จำเป็น |

## 5) Research Questions → ส่งต่อ Reese

### กลุ่ม 1 — คู่แข่ง (อัปเดตจาก Idea Card เดิม)
1. เครื่องมือทำเกียรติบัตรหมู่ที่ครู/HR ไทยใช้จริงตอนนี้: autoCrat, Canva bulk create, เว็บไทยเจ้าไหนบ้าง? แต่ละตัวรับ input แบบไหน (บังคับ CSV? Google Sheets?) และมีตัวไหน **รับ .docx/.pdf ตรง ๆ** แบบเราไหม?
2. Canva bulk create อยู่ใน tier ไหน ราคาเท่าไหร่ และ template เกียรติบัตร "สไตล์ราชการ/สถาบันไทย" บน Canva มีมาก-น้อยแค่ไหน? (ทดสอบสมมติฐานจุดต่างของเรา)
3. มีเว็บฟรีของไทย (เช่นของ สพฐ./ศึกษานิเทศก์/ครูทำแจกกัน) ที่ทำเรื่องนี้อยู่แล้วไหม — ถ้ามี จุดอ่อนที่ผู้ใช้บ่นคืออะไร?

### กลุ่ม 2 — Lib client-side สำหรับอ่านไฟล์ (ตัดสินสถาปัตยกรรมโดยตรง)
4. อ่าน **.docx** ในเบราว์เซอร์: mammoth.js หรือทางเลือกอื่น — ตัวไหนดึง "รายการชื่อ" จากทั้งย่อหน้าและตารางใน Word ได้ดีสุด? ข้อจำกัดกับภาษาไทย?
5. อ่าน **.xlsx**: SheetJS (xlsx) community edition ยัง maintain อยู่ไหม license เป็นอะไร มีทางเลือกเช่น exceljs ที่เบากว่าสำหรับแค่อ่านค่า cell ไหม?
6. อ่าน **.pdf**: pdf.js ดึง text ภาษาไทยได้แม่นแค่ไหน (ปัญหาสระ/วรรณยุกต์เรียงเพี้ยน, PDF ที่เป็นภาพสแกน)? ควรตั้ง expectation ว่า PDF รองรับเฉพาะแบบ text-based ใช่ไหม?
7. ทุก lib ข้างต้นรันบน GitHub Pages (static, no server) ได้ 100% ไหม มี bundle size รวมเท่าไหร่ กระทบเวลาโหลดบนมือถือครูแค่ไหน?

### กลุ่ม 3 — ฟอนต์ไทยตอน generate PDF (ความเสี่ยงเทคนิคอันดับหนึ่ง)
8. Lib สร้าง PDF ฝั่ง client ตัวไหนจัดการ**ฟอนต์ไทยฝังในไฟล์ + การตัดคำ/สระบน-ล่าง**ได้ถูกต้อง: jsPDF (+ฟอนต์ custom), pdf-lib, pdfmake, หรือแนว render HTML/SVG→PDF (เช่น print CSS / html2canvas)? ขอคำแนะนำ 1 ตัวพร้อมเหตุผลและตัวอย่างที่พิสูจน์แล้วกับภาษาไทย
9. ฟอนต์ไทยที่ license อนุญาตให้**ฝังใน PDF เพื่อการค้า**ได้ฟรี: ชุดฟอนต์แห่งชาติ (TH Sarabun New ฯลฯ), Google Fonts ไทย (Sarabun, Chonburi, Charm...) — ตัวไหนเหมาะกับงานเกียรติบัตร (หัวเรื่องแบบอาลักษณ์/ตัวเนื้อทางการ) และไฟล์ฟอนต์ใหญ่แค่ไหนเมื่อฝังทั้งชุด?
10. เกียรติบัตรมีชื่อคนซึ่งบางครั้งมียศ/คำนำหน้ายาว ("ว่าที่ร้อยตรีหญิง...") — มี edge case การ render ไทยที่ lib PDF มักพลาดไหม (สระลอย, ญ ฐ ตัดหาง)?

### กลุ่ม 4 — ขนาด/สเปกเกียรติบัตรมาตรฐานไทย
11. เกียรติบัตรไทยมาตรฐานใช้ขนาดอะไรบ้าง: A4 แนวนอนคือหลักใช่ไหม สัดส่วนการใช้แนวตั้ง/A5 มีนัยยะไหม? ระเบียบ/ธรรมเนียมราชการเรื่อง layout (ตราครุฑ? ตราหน่วยงาน? ตำแหน่งผู้ลงนามซ้าย-ขวา?) มีเอกสารอ้างอิงไหม?
12. สเปกงานพิมพ์จริงที่ควรเผื่อใน artwork ตั้งแต่แรก: bleed เท่าไหร่, ระยะขอบปลอดภัย, 300dpi ที่ A4 — เพื่อให้ไฟล์จากเว็บ**ส่งเข้าแท่นพิมพ์ TANAPAT ได้เลย**โดยไม่ต้องทำใหม่ (นี่คือหัวใจ upsell)
13. การใส่ตราโรงเรียน/ตราครุฑ/ลายเซ็นผู้บริหารในเกียรติบัตรที่ generate จากเว็บ มีระเบียบราชการหรือข้อกฎหมายที่ต้องเตือนผู้ใช้ไหม? (ยกมาจาก Idea Card เดิม — ยังไม่มีคำตอบ)
